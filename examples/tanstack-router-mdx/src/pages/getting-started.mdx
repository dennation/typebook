# Getting Started

Welcome to the **UI Studio** MDX example. This page demonstrates an `.mdx` file
acting as a route component — markdown text with live React components inline.

## Live component

Click this button (it's a real React component embedded mid-paragraph):

<div style={{ padding: '12px 0' }}>
  <Button>Click me</Button>
</div>

## Sizes

The `<Button>` component supports three sizes: `sm`, `md`, and `lg`.

<div style={{ display: 'flex', gap: 8, padding: '12px 0' }}>
  <Button size="sm">Small</Button>
  <Button size="md">Medium</Button>
  <Button size="lg">Large</Button>
</div>

## Variants

- **solid** — filled background
- **outline** — bordered, transparent background
- **ghost** — no border, no background
