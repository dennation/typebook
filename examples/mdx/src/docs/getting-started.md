# Getting Started

Welcome to the **UI Studio** MDX example. This page demonstrates how to use
markdown files for documentation with live React components alongside.

## Using the Button component

The `<Button>` component supports three sizes: `sm`, `md`, and `lg`.

## Variants

Buttons support three visual variants:

- **solid** — filled background
- **outline** — bordered, transparent background
- **ghost** — no border, no background
