# Changelog

## v0.2.0

- Added `<Story />` component for embedding stories in documentation pages
- Added `definePage()` API for standalone documentation
- Added MDX/markdown support via `@mdx-js/rollup`

## v0.1.0

- Initial release
- `define()` API with `single()`, `variants()`, `matrix()` story types
- Vite and Webpack plugin support
- TypeScript prop extraction with auto-generated variant grids
- Sidebar navigation with path grouping
- Dark/light theme support
- Interactive docs page with prop controls
